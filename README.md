<div id="titulo" align="center">
    <h1 align="center">Control de Horario</h1>
</div>

---

<div id="Caracteristicas" align="center">
    <h2 align="center"> 🛠️ Especificaciones:</h2>
    <h3>
        Este proyecto fue creado con fines de practica.<br>
        Desarrollado en HTML, CSS y javaScript.<br>
        Este programa busca facilitar la carga de horas de una persona, <br>
        calculando horas extras, feriados, sábados y domingos, devolviendo una <br> 
        cantidad estimada de horas trabajadas y su correspondiente remuneración.
    </h3>
</div>

---

<div id="datos" align="center">
    <h2>💾 Datos:</h2>
    <br>
        Permite a las personas que trabajan por hora/quincena llevar un control de horas.<br>
        Carga de año, mes y quincena actualizada.<br>
        Carga de horario de entrada y salida.<br>
        Permite elegir dias feriados.<br>
        Carga de valor por hora trabajada.<br>
        Retorna un valor estimado de la remuneracion neta bruta a cobrar.<br>
        Totalmente responsivo.<br>
</div>
